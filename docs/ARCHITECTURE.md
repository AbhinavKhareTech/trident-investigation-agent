# System Architecture

## Overview

The Trident Investigation Agent is a two-layer system:

1. **Trident Engine** вҖ” Production-grade AI agent infrastructure (context engineering, FSM, lineage, reliability)
2. **Investigation Agent** вҖ” Domain-agnostic autonomous investigation pipeline built on top of the engine
